# 놀러와 박물관 [Play Museum]

국내 박물관 전체 정보를 확인 할 수 있는 홈페이지가 부족하다.

* 국내 박물관 정보를 지역별로 찾아 볼 수 있다
* 가고 싶은 박물관을 북마크하여 저장 할 수 있다.
* 방문 후기를 작성 할 수 있다.

* 추후 Membership 서비스로 북마크, 방문 후기를 서버에서 서비스 할 수 있도록 할 예정이다.
